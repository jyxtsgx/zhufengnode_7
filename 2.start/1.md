## 基础
1. 所有在path目录下的内容都可以全局使用例如(node npm)
2. 查看node版本

```javascript
node -v
```
会自动安装npm 
```javascript
npm -v
```

## mac安装node可以直接默认下一步即可
```javascript
brew install node 直接安装
```
